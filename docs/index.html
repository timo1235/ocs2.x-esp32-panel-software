ESP32-Panel Firmware installer

The installer can be used to install the ESP32_Panel firmware to an ESP32 without the need of an IDE or compiling the source code. The firmware works in combination with the OPEN-CNC-Shield 2 Firmware. Just connect the ESP32 with the computer and click on "Connect".

Requirements

  • ESP32 Panel PCB with an ESP32Devkit V1 or
  • ESP32 Panel Integrated Battery with an ESP32Devkit V1
  • Google Chrome Browser - others might also work with WebSerial
  • Maybe driver for your PC -> google for ESP32 driver and install them

First Step

Connect the ESP32 to your computer and click on the "Connect" button below. Choose the serial port of the ESP32 and then install the firmware

If the ESP32 is not recognized, you may need to install the driver for the ESP32. Google for "ESP32 driver" and install it.

Connect to the ESP32

After the firmware installation, the ESP32 will be configured as an access point called "ESP32-Panel-XXXXXX".

Connect to the access point with your computer or smartphone and open this link in the broswer: https://192.168.4.1. You should see the configuration webinterface.

If a mobile phone is used make sure that the mobile data is disabled and the WLAN is connected to the right network. Mobile phones can be picky about the network if the dont have internet access.

Configure using the Webinterface

Configure the ESP32 with the settings you need and click on "Reboot". The ESP32 will restart the settings will be active

The settings will only be effective after a reboot.

Der Installer kann verwendet werden, um die ESP32_Panel Firmware auf einem ESP32 zu installieren, ohne dass eine IDE oder das Kompilieren des Quellcodes erforderlich sind. Die Firmware funktioniert in Kombination mit der OPEN-CNC-Shield 2 Firmware. Verbinde einfach den ESP32 mit dem Computer und klicke auf "Verbinden".

Voraussetzungen

  • ESP32 Panel PCB mit einem ESP32Devkit V1 oder
  • ESP32 Panel mit integriertem Akku mit einem ESP32Devkit V1
  • Google Chrome Browser - andere könnten auch mit WebSerial funktionieren
  • Vielleicht Treiber für deinen PC -> suche nach ESP32 Treiber und installiere sie

Erster Schritt

Verbinde den ESP32 mit deinem Computer und klicke unten auf den "Verbinden"-Knopf. Wähle den seriellen Port des ESP32 und installiere dann die Firmware

Wenn der ESP32 nicht erkannt wird, musst du möglicherweise den Treiber für den ESP32 installieren. Suche nach "ESP32 Treiber" und installiere ihn.

Verbindung zum ESP32

Nach der Installation der Firmware wird der ESP32 als Zugangspunkt mit dem Namen "ESP32-Panel-XXXXXX" konfiguriert.

Verbinde dich mit dem Zugangspunkt über deinen Computer oder dein Smartphone und öffne diesen Link im Browser: https://192.168.4.1. Du solltest das Konfigurationswebinterface sehen.

Wenn ein Mobiltelefon verwendet wird, stelle sicher, dass die mobilen Daten deaktiviert sind und das WLAN mit dem richtigen Netzwerk verbunden ist. Mobiltelefone können wählerisch bezüglich des Netzwerks sein, wenn sie keinen Internetzugang haben.

Konfigurieren über das Webinterface

Konfiguriere den ESP32 mit den Einstellungen, die du benötigst, und klicke auf "Neustarten". Der ESP32 wird neu gestartet und die Einstellungen werden aktiviert

Die Einstellungen werden erst nach einem Neustart wirksam.
